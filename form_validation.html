Form Validation and Attributes

Form Validation

Form Validation is the process of making sure that the data submitted through web forms is accurate, consistent and conforms to specific rules and requirements that are predefined.

Explaining the Attributes

Minlength Attributes specify the minimum values for and input field.

Pattern Attributes specifies a regular expression that the input field's value is checked against, when the form is submitted.

Required attribute specifies that an input field must be filled out before submitting the form.
